Architektura MicroBlink 3M

3 Miliony Parametrů

Mikroskopická velikost umožňující okamžitou odezvu a plný provoz v paměti VRAM běžných přenosných počítačů bez ztráty přesnosti.

0% UNK Tokenizer

Unikátní, bezpečně uzamčená česká abeceda a BPE slovník zaručující nulový výskyt neznámých tokenů [UNK] při zpracování diakritiky.

Grouped-Query Attention

Implementace GQA (známé z velkých modelů Llama 3) drasticky snižuje paměťové nároky KV cache a zrychluje generování textu na CPU.

Integrovaný RAG Web-Search

Architektura je od základu navržena pro spolupráci s internetovým kontextem a přesné extrahování faktů bez halucinací.

Ukázka inicializace

// Inicializace modelu MicroBlink 3M v Pythonu
# 1. Instalace open-source knihovny
$ pip install openblink-ai

# 2. Načtení architektury s GQA a tokenizéru
import openblink
model = openblink.load("microblink-3m")

# 3. Spuštění lokální inference s internetovým RAG
$ python run_model.py --initialize --rag_enabled
