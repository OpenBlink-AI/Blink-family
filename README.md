# ⚡ OpenBlink: TinyBlink-14M

TinyBlink-14M je extrémně lehký, rychlý a 100% open-source český jazykový model (LLM) s pouhými 14 miliony parametrů. Projekt vznikl s cílem dokázat, že efektivní AI se dá trénovat s minimálními náklady a může běžet lokálně na jakémkoliv běžném hardwaru.

## 🚀 Hlavní vlastnosti
- **Velikost:** 14M parametrů (vhodné pro mobilní telefony, slabé PC nebo Raspberry Pi)
- **Jazyk:** Čeština 🇨🇿
- **Architektura:** MiniGPT (GPT-2 style)
- **Licence:** Apache 2.0 (zcela zdarma pro komerční i nekomerční využití)
