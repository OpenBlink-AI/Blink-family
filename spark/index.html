1. Detailní Anatomie & Architektura Modelu

Blink Spark 12M je nejmenším členem rodiny OpenBlink v1.0 "Andromeda". Vznikl jako reakce na potřebu bleskové předfiltrace uživatelských vstupů předtím, než jsou odeslány do drahých a pomalých velkých LLM (jako GPT-4o nebo Claude).

Model využívá vysoce komprimovanou architekturu **Decoder-Only Transformer** s optimalizovaným Attention mezipamětí. Díky striktní redukci počtu vrstev a optimalizovanému vocab-size (32 000 českých tokenů) se kompletní model v GGUF Q4_K_M kvantizaci vejde do pouhých **28 MB na disku**!

Klíčové Parametry Architektury:

  • Počet vrstev (Hidden Layers): 12
  • Hidden Dimension ($d_{model}$): 384
  • Attention Heads: 6
  • Kontextové okno: 2048 tokenů
  • Tokenizer: Nativní český Byte-Pair Encoding (BPE) trénovaný na 1.2M českých dokumentů.

2. Hlavní Případy Užití (Use-Cases)

Vzhledem ke své velikosti 12M neslouží Spark k psaní dlouhých slohových prací. Jeho síla spočívá v **okamžité analitické klasifikaci textu**:

  • Intent Router / Guardrail: Rozpozná, zda uživatelský dotaz vyžaduje RAG vyhledávání, kalkulaci, nebo zda jde o pozdrav. Ušetří až 80 % volání velkého LLM.
  • Detekce Spamu & Toxicidy v Češtině: Bleskový filtr pro diskuzní fóra a e-shopy. Analýza 1 000 komentářů zabere na CPU méně než sekundu.
  • IoT & Smarthome: Běží přímo na mikrokontrolerech a lokálních serverech bez internetového připojení. Porozumí příkazům jako *"Zhasni v obýváku"* s latencí pod 5 ms.

3. Rychlá Integrace a Příklady Kódu

A. Běh v Pythonu přes HuggingFace Transformers

Python API pip install transformers torch
# Načtení Blink Spark 12M z HuggingFace Hubu from transformers import AutoModelForCausalLM, AutoTokenizer model_id = "openblink/blink-spark-12m" tokenizer = AutoTokenizer.from_pretrained(model_id) model = AutoModelForCausalLM.from_pretrained(model_id) prompt = "Klasifikuj záměr uživatele: [Chci reklamovat nefunkční boty] ->" inputs = tokenizer(prompt, return_tensors="pt") outputs = model.generate(**inputs, max_new_tokens=10) print(tokenizer.decode(outputs[0], skip_special_tokens=True)) # Výstup: ZÁMĚR: REKLAMACE
